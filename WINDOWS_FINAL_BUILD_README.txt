FINAL WINDOWS BUILD

1. Keep a backup of the current database and installed application.
2. Install Node.js LTS.
3. Install Rust using rustup.
4. Install Microsoft Visual Studio Build Tools with Desktop development with C++.
5. Double-click BUILD_WINDOWS_FINAL.bat.
6. The installer will be generated under src-tauri\target\release\bundle\.

AUTO UPDATE BLOCKER
The updater endpoint and public signing key are still placeholders. A normal installer can be built and tested, but production automatic updates require real GitHub repository details and Tauri signing keys.
