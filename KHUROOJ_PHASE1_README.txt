Nabd Al-Gharb Desktop - Khurooj Phase 1

Added:
1. Khurooj card on the main desktop screen.
2. Employee-linked Khurooj records.
3. Add, edit, delete and mark returned actions.
4. Exit Re-entry, Final Exit, Business Travel and Other types.
5. Exit date, return date, visa expiry, airport, remarks and attachment URL.
6. Automatic Active, Returning Soon, Overdue, Returned and Final Exit status.
7. Iqama and passport number/expiry fields in Employee module.
8. Supabase CRUD and realtime synchronization.
9. Return-date and document-expiry reminder summary.

Required setup:
Open Supabase SQL Editor and run SUPABASE_KHUROOJ_MODULE.sql once.

Reminder rule:
The Khurooj screen shows reminders when the return date, Iqama expiry or passport expiry is within 30 days, or already overdue.
